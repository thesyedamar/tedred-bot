# Tedred Bot

Tedred Bot is a support and lead-generation bot project built with Python. It combines backend logic, data handling, and a lightweight chat widget to support customer conversations from a website experience.

## Highlights

- Python bot backend for support conversations
- Chat widget interface for website embedding
- Lead-generation and customer-support workflow
- Deployment-ready structure with Render configuration

## Tech Stack

- Python
- HTML chat widget
- Render deployment configuration
- Spreadsheet/data integration helpers

## Run Locally

```bash
pip install -r requirements.txt
python main.py
```

## Project Focus

This project demonstrates automation for customer communication: capturing user messages, supporting basic bot workflows, and preparing a website-friendly chat experience.
